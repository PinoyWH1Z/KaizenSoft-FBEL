KaizenSoft-FBEL

Facebook Easy Login



With this Dynamic Link Library (DLL), you can easily add a facebook login/auth in your application/software. This is more-like a class by which helps you configure your software to adopt the way "Facebook" uses to log-in their users.

How to use?:

Simply add "KaizenSoft - FBEL.dll" as your project reference and use the guides/examples below to make it functional.

Example usage in C#:

KaizenSoftFBEL.KaizenSoftFBEL fbLogin = new KaizenSoftFBEL.KaizenSoftFBEL();

if (fbLogin.Enter(textBox1.Text, textBox2.Text))

{ MessageBox.Show("Login Successful"); }

else

{ MessageBox.Show("Login Error."); }

Example usage in VB.NET:

Dim fbLogin As New KaizenSoftFBEL.KaizenSoftFBEL()

If fbLogin.Enter(TextBox1.Text, TextBox2.Text) Then

MessageBox.Show("Login successful.")

Else

MessageBox.Show("Login error.")

End If
